Load a Data Set

This section will let you load a specific data set

After requesting, the Introduction tab in the Main view will be automatically switched to the tab containing the requested data set

  • Side bar contains the requested data set's Navigation button
  • Main View contains the contents of the data set
  • Tab header contains the file names of the data sets
  • Chat window will have responses for the requested user query

Run a Bar Graph visualisation on the data set

This is an interactive bar chart used for visual-based filtering. It allows the user to filter by categories. The graph is displayed in the Main view depending on how the data is sorted

Let us explain it more clearly using an example,
User: I want a bar graph visualisation for the current table
Assistant: ok, before proceeding tell me what is the x-axis
User: City
Assistant: Great! and which feature should be along y-axis ?
User: Wine
Assistant: your requested bar graph is now added to the main view


Run a Force Directed Graph visualisation on the data set

This is an interactive force directed graph used for visualisation of data. Force-directed graph drawing assigns forces among the set of edges and the set of nodes of a graph drawing


Below is an illustration in detail,
User: I want force directed graph visualization
Assistant: ok, before proceeding tell me what is the source node
User: city1
Assistant: Great!, and which feature should be the target node ?
User: city2
Assistant: which feature should depict the strength between the nodes ?
User: strength is distance
Assistant: your requested bar graph is now added to the main view
