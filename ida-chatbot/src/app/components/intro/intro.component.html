Load a Data Set

This section will let you load a specific data set

After your request to load a data set, the requested data set will be added as a new item in the sidebar and the data-tables inside it will be loaded to the main-view as separate tabs

  • Side bar contains the requested data set's Navigation button
  • Main View contains the contents of the data set
  • Tab header contains the file names of the data sets
  • Chat window will have responses for the requested user query

Let us explain it more clearly using an example,
User: I would like you to load the city dataset
Assistant: Data set is now loaded. You can access it in the Main view

Run a Bar Graph visualisation on the data set

User can request a BarChart visualization for the loaded data-tables. IDA also allows user to define a filtering criteria on the data for which the BarChart is requested

Let us explain it more clearly using an example,
User: I want a bar graph visualisation for the current table
Assistant: ok, before proceeding tell me what is the x-axis
User: City
Assistant: Great! and which feature should be along y-axis ?
User: Wine
Assistant: Great! and How many items would you like to see?
User: Top 10 records, sorted descending on wine
Assistant: Your requested bar graph is now added to the main view


Run a Force Directed Graph visualisation on the data set

This is an interactive force directed graph used for visualisation of data. Force-directed graph drawing assigns forces among the set of edges and the set of nodes of a graph drawing


We will walk through in detail with an example,
User: I would like a force directed graph visualization for the current table
Assistant: Ok, before proceeding tell me what is the source node
User: Source node is city1
Assistant: Great!, and which feature should be the target node ?
User: Target node is city2
Assistant: Which feature should depict the strength between the nodes ?
User: Strength between the nodes should be represented by distance
Assistant: Your requested bar graph is now added to the main view


Classify each data point into a specific group through Clustering

User can request the IDA to run a clustering algorithm of their choice on the loaded data set, IDA will provide the user with available algorithms for clustering and collect the parameters required to run the algorithm


Let us see how this works effectively with an example,
User: What are the available clustering algorithms?
Assistant: Here are the list of algorithms, please select one to proceed [Hint] - K-means - Affinity Propogation
User: I would like to run the KMeans algorithm on the current table
Assistant: Following is the list of parameters for " kmeans " :

  1. init [Optional]
  2. n_init [Optional]
  3. max_iter [Optional]
  4. precompute_distances [Optional]
  5. verbose [Optional]
  6. random_state [Optional]
  7. copy_x [Optional]

Which optional parameters you want?
[HINT : Optionals are .......]
User: Optional parameters should be init, n_init and tol
Assistant: Your params are:
  1. init Type : k-means++, random, an ndarray
  2. n_clusters Type : int
  3. n_jobs Type : int
Enter values for the parameters [HINT : Set parameter as ...]
User: Set n_clusters as 5
Assistant: Value Stored, Enter next Value
  1. n_jobs Type : int
  2. init Type : int
  3. n_init Type : int
  4. n_tol Type: float

User: Set n_jobs as 8
Assistant: Value Stored, Enter next Value
  1. init Type : int
  2. n_init Type : int
  3. n_tol Type: float

User: Set init as random
Assistant: Value Stored, Enter next Value
  1. n_init Type : int
  2. n_tol Type: float

User: Set n_init as 5
Assistant: Value Stored, Enter next Value
  1. n_tol Type: float

User: Set tol as 3
Assistant: All the values have been taken, Enter the clustering features
User: Clustering features are wine, cinema and gasoline
Assistant: Which Column is Key Label Feature?
User: Label feature should be city
Assistant: Clustered data is displayed! You can access it in the Main view
